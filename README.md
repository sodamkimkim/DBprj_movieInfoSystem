## **기능**

버튼을 이용한 DATA CRUD

SELECT, CREATE, UPDATE, DELETE 기능을 구현

## **기술**
> JAVA SWING 을 이용한 화면 랜더링
> 
> 
> Dto, Dao, Service 설계
> 
> DB Table 연관 관계 설정
> 
> Query문을 통한 DB 조작
>
